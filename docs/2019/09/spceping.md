纯双拼测评系统可以统计双拼方案的按键频率、异手率（互击率）、音节当量、连续当量、词组当量、词组韵声当量等数据。

## 使用

导入全拼到双拼的映射表，点击**按钮**选择双拼。

## 说明

音节当量 = $\Sigma$(音节当量\*音节频率)  
音节累计当量 = $\Sigma$(音节当量\*音节累计频次)
